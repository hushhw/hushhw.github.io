>
> * vs code 常用快捷键（windows）
> * 常用的插件
> * 使用心得

​          

## 快捷键

* 格式化代码
  * windows ：`Shift + Alt + F`

​           

## 插件

* **翻译(translate to chinese)**：使用百度翻译API把其他语言翻译成中文，快捷键是 `ctrl+shift.t`。
